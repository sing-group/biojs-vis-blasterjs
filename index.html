How to use

Requirements

Bootstrap 3:
Avaliable on http://getbootstrap.com/

Html2canvas:
Avaliable on https://html2canvas.hertzen.com/

Accepted files

BLAST output files -outfmt 0 (Default):
Only one query on the BLAST file
You can check NCBI BLAST user manual

<head>
    ...
    <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.6/css/bootstrap.min.css" integrity="sha384-1q8mTJOASx8j1Au+a5WDVnPi2lkFfwwEAa8hDDdjZlpLegxhjVME1fgjWPGmkzs7" crossorigin="anonymous" />

</body>

About BlasterJS

BlasterJS is a free to use, open source JS library created by SING Group.

© SING - Sistemas Informáticos de Nueva Generación 2016

Display hits distribution

Display BLAST hits distribution in the query sequence. Clicking on one hit, you can see all measures on a table. Using the "Download as image" button you can download the distribution plot as PNG.

Display data on table

All hits are displayed with all their measures on a table. Clicking on one description you can see this alignment. You can download it as PNG image or as CSV file.

Display alignments singly

For all hit, you can visualize the singly alignment. This alignment can be downloaded as PNG image.

Change alignment colors

You can show this image full-colored or in gray-scale. You can change it clicking on "Click to change colors to" button.

Save images and tables

All images can be downloaded as PNG image. In addition, the table can be downloaded as CSV file.

Use it everywhere

You can use BlasterJS on any web page. In addition, you can select which elements you want to display. In this image, user only display the distribution of hits image and the table.
